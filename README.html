プライバシーポリシー

本アプリ「mennikki」は、氏名、メールアドレス、位置情報、端末識別子など、個人を特定可能な情報を収集しません。

データの取り扱い

  • 入力されたデータ（ラーメンの記録など）は端末内にのみ保存されます。
  • 本アプリは、ユーザーのデータを外部サーバーへ送信しません。
  • 写真ライブラリへのアクセスは、ユーザーが選択したラーメン画像をアプリ内の記録として端末内に保存する目的のみに使用します。
  • 広告表示や第三者へのデータ提供は行いません。

プライバシーポリシーの変更

本ポリシーの内容を変更する場合は、本ページにて更新いたします。

App Store のプライバシー表示との整合性

本ポリシーは、App Store に表示されるプライバシー情報と一致しています。
